Galactic Civilizations Spread

Degree Project ‚Ä¢ Computational Modeling

Simulating how civilizations could spread through the galaxy via stellar encounters

Click to explore ‚Üí

Project Overview

This degree project explores the Fermi Paradox by simulating civilization spread through the galaxy. Using the epicycle approximation to track star motion, the model investigates colonization patterns under various parameters.

Simulation Parameters

  • Examined 200-1000 Sun-like stars with suitable planets
  • Modeled civilization development via probability distributions
  • Tracked stellar dynamics over 12+ billion years
  • Analyzed colonization exponential growth patterns

Results

  • First advanced civilization after 0.27-4.8 Gyr depending on parameters
  • Single spreading civilization dominated with 89% colonization (1000 stars, N=3)
  • Stationary civilizations dominated with 99-99.7% colonization when isolated
  • Mean nearest neighbor distance: 36-82 pc depending on star count

Implications

Results suggest that increasing stellar density combined with favorable evolution parameters could enable rapid galactic colonization by a single civilization, potentially addressing aspects of the Fermi Paradox.

Project Focus

Advanced 3D magnetohydrodynamic simulations exploring the magnetic coupling between young host stars and their orbiting exoplanets, crucial for understanding habitability and atmospheric evolution.

Simulation Techniques

  • 3D MHD code implementation (PLUTO/Athena)

  • High-performance computing on supercomputers
  • Data analysis with Python and Tecplot

Scientific Insights

  • Magnetic disconnection structures
  • Energy transfer mechanisms
  • Atmospheric escape rates
  • Exoplanet habitability assessment

Computational Performance

Utilized large-scale HPC resources with Fortran optimization achieving significant speedups in simulation execution.
